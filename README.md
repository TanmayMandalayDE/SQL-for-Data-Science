# Sales Performance & Customer Insights Analysis Using SQL

## Overview

This project analyzes retail and e-commerce sales data for a multi-city business operating across India. The objective is to identify the drivers of revenue, profit, customer retention, and regional growth using SQL-based analysis that mirrors real business reporting.

The project is designed around practical questions that hiring managers expect analysts to solve: Which products and categories generate the most revenue and margin? Which cities and channels are growing or slowing down? How valuable are repeat customers over time? Which customers are at risk of churning? To answer these, I built a relational sales database and used advanced SQL techniques including CTEs, window functions, joins, subqueries, conditional aggregation, cohort analysis, and customer lifetime value estimation.

This is not a toy dataset project. It is structured like a real commercial analytics workflow, where raw transactional tables are transformed into insights that support pricing, inventory planning, regional expansion, marketing allocation, and retention strategy. The output can also be connected to Power BI or Tableau to create stakeholder dashboards.

This project demonstrates both technical SQL capability and business judgment, especially in sales, revenue, and operations reporting.
